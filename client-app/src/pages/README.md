# pages/ — заготовка миграции на react-router-dom

**НЕ ПОДКЛЮЧЕНО к main.tsx** — все файлы в этой директории и в `components/ui/` являются заготовкой для будущей миграции с монолитного `App.tsx` на роутер-архитектуру.

- `main.tsx` рендерит только `<App />` (без роутера)
- `react-router-dom` есть в зависимостях, но не используется
- Живой путь: `components/*` (top-level)

**Статус на 14.07.2026:** обе версии (pages/* + components/ui/* и живой App.tsx + components/*) прошли аудит и содержат фиксы Модуля 8. При будущем подключении роутера требуется полный тест паритета фичей между двумя путями.

Не удалять без сверки, что всё перенесено в живой путь. Не подключать без полноценного QA.
